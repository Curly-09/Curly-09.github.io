About meSairaj

Hi, I'm Sairaj Ambekar, a 16-year-old developer from Mumbai, India. For the past year I've been building Sylix, a free-to-run native AI code editor written from scratch in Rust.

I built the whole thing solo: the native editor, the backend, the API layer, the model routing, and the verify loop. Most AI coding tools just write code. Sylix is built to understand the project, remember your rules, test what it writes, and actually finish the task. I cold-DMed 100+ developers to figure out what a real AI editor should do, then built it.
